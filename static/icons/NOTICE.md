# Third-party assets

The SVG icons in this directory are sourced from the LEGO Education
SPIKE Prime web IDE (https://spike.legoeducation.com/prime/project) and
remain the intellectual property of **The LEGO Group**.

They are reused here for **educational fair-use** purposes only — to
provide First LEGO League (FLL) students with a familiar block-coding
visual language while practising offline. This project is **not
sponsored, endorsed, or affiliated with** The LEGO Group, LEGO
Education, or the FIRST organisation.

LEGO®, SPIKE™ and the brick configuration are trademarks of The LEGO
Group.

If you are an FLL coach distributing this tool to students, you are
encouraged to point them at the official LEGO Education materials at
https://education.lego.com/ as the primary learning resource. If you
represent The LEGO Group and would like these assets removed or
re-licensed, please open an issue on the project's tracker.

The two extracted-from-base64 emblems (Motors.svg, Movement.svg, Hub.svg,
UltraSound.svg, EventsStart.svg, ColorSensor.svg, ForceSensor.svg) have
been re-coloured to white (#FFFFFF) so the standard Blockly category
backgrounds provide the per-category tint, matching the LEGO IDE's
runtime presentation.
